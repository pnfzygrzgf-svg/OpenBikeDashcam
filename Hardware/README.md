# Komponenten: 
<img src="Anschlussschema.jpg" alt="Anschlussschema der Dashcam Komponenten" width="600">  

1) Raspberry Pi cm5 (das compute module, wegen der Baugröße. Der Protoyp läuft auf der Version 8GB RAM mit 32GB emmc, aber ein kleiner sollte auch reichen)
2) Waveshare CM5 Nano-IO Board (A) - (also nur USB-C und USB-A anschluss)
3) Raspberry pi camera module 3 wide
4) 2 x rgb led modul für die Status- und Rücklicht LED (z.B. ky-016 FZ0455)
5) 4 x Widerstände für die LEDs (z.B. 330 Ohm) 
6) Sparkfun XM125 A121
7) USB Stick (möglichst USB 3)
8) RTC Batterie RPI-RTCBAT von Berry Base
9) ggf. Powerbank mit mind. 5V, 5A oder ein passendes Netzteil

## Zubehör
10) qwiic kabel (qwiic stecker female auf Dupont stecker female)
11) 6 jumper Kabel (female)
12) USB Kabel für die Powerbank
13) Kamerakabel ( 22-polig/0,5 mm Pitch zu 15-polig/1,0 mm Pitch)
